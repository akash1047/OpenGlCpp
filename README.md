# OpenGlCpp

## build
### get the source code

> :warning: do not download the zip!!!

- required: `git`

in terminal
```
    git clone --recursive -j2 https://github.com/akash1047/OpenGlCpp.git
```

### cmake build
navigate to directory `OpenGlCpp`
Then in terminal run these commands.

#### build & run
```
    mkdir build
    cd ./build
    cmake ..
    cmake --build .
    ./ogc
```
